SEO Slider Auditor
==================

Qué hace
--------
Extensión de Chrome con panel lateral para realizar auditorías SEO básicas y técnicas sobre la página web activa, mostrando resultados en tiempo real mediante un sistema de puntuación ponderada.

La herramienta analiza elementos de SEO on-page, accesibilidad, metadatos y señales técnicas de la página.

Importante
----------
No todo puede verificarse automáticamente desde una extensión del navegador. Por ello, la auditoría distingue entre tres tipos de comprobaciones:

- Automático → Verificación directa desde el código HTML.
- Heurístico → Aproximaciones basadas en buenas prácticas SEO.
- Manual → Revisión recomendada por el usuario.

Mejoras implementadas
---------------------
Se han añadido mejoras en el análisis SEO y en la experiencia de usuario, incluyendo:

- Detección y validación de meta description:
  - Verifica existencia.
  - Evalúa longitud óptima entre 120 y 160 caracteres para SEO.

- Análisis de Open Graph:
  - Verificación de og:title.
  - Verificación de og:description.
  - Mejora la optimización para compartir contenido en redes sociales.

- Mejora del sistema de puntuación SEO:
  - Sistema ponderado según la importancia de cada factor.
  - Feedback visual con estados:
    - Cumple
    - Mejorable
    - Error
    - Manual

- Mejoras en la interfaz de usuario:
  - Diseño moderno tipo dashboard.
  - Efectos visuales suaves.
  - Animaciones en la carga de resultados.

Checks incluidos
----------------
La extensión analiza los siguientes factores:

- Jerarquía de encabezados H1-H3
- Extensión del contenido (thin content)
- Densidad de enlaces internos y externos
- robots.txt
- sitemap.xml
- Google Analytics / Google Tag Manager
- Manejo de errores 404 (revisión heurística)
- Accesibilidad básica (atributos ALT, lenguaje del documento)
- Redes sociales básicas
- Meta verificación de Bing Webmaster Tools
- Meta description SEO
- Open Graph tags
- Presencia en buscadores (revisión manual)
- Google Trends (revisión manual)
- Indexación de sitemap (revisión manual)

Cómo cargar la extensión
------------------------
1. Abre: chrome://extensions/
2. Activa el "Modo de desarrollador".
3. Pulsa "Cargar descomprimida".
4. Selecciona la carpeta del proyecto.
5. Haz clic en el icono de la extensión para iniciar el análisis.

Notas
-----
- Algunas webs pueden bloquear parte del análisis por políticas de seguridad.
- La detección de SEO técnico es orientativa y debe complementarse con herramientas profesionales.
- La indexación real en Google o Bing requiere herramientas oficiales de webmaster.
