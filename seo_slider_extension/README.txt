SEO Slider Auditor
==================

Qué hace
--------
Extensión de Chrome con panel lateral para revisar una URL y mostrar si cumple una serie de checks SEO / técnicos / de accesibilidad.

Importante
----------
No todo puede verificarse automáticamente desde una extensión. Por eso la herramienta distingue entre:
- Automático
- Heurístico
- Manual

Checks incluidos
----------------
- Más páginas que index
- robots.txt
- sitemap.xml
- señal de Google Analytics
- manejo de 404 / redirección
- enlazado interno y externo
- accesibilidad básica
- LinkedIn
- meta de Bing Webmaster Tools
- accesos rápidos para revisar Google/Bing/Google Trends

Cómo cargarla
-------------
1. Abre chrome://extensions/
2. Activa Modo de desarrollador
3. Pulsa Cargar descomprimida
4. Selecciona esta carpeta
5. Pulsa el icono de la extensión

Notas
-----
- Algunas webs pueden bloquear parte del análisis.
- La comprobación de Google Analytics, anuncios o accesibilidad es orientativa.
- Google Indexación, Bing Webmaster real o pruebas en varios navegadores requieren revisión manual.
